Requirements
==============

- JRuby 1.4.0
- Rubygems
    - Rake
    - Thor
- Cappuccino Tools

    - sudo ./bootstrap.sh   (installs narwhaljs and support tools)
    - jake sudo-install
- Titanium SDK (Install Titanium Developer)
- Python
